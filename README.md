# Excel Refresh 프로젝트

## 목적
여러 엑셀 파일들의 데이터를 자동으로 새로고침하는 프로그램

## 파일 구조

### 메인 파일
- **main.py**: 전체 작업 실행 진입점
- **excelrefresh_time_delay.py**: 엑셀 리프레시 핵심 로직

### 백업 파일 (현재 미사용)
- **xlsmtodb_backup.py**: 엑셀 → CSV/DB 변환 기능
- **asset_backup.py**: 개발행위허가 CSV → DB 변환 기능

## 사용 방법

```bash
python main.py
```

## 실행 순서
1. main.py 실행
2. excelrefresh_time_delay.py가 자동으로 실행됨
3. 지정된 엑셀 파일들을 순차적으로 리프레시
4. 마지막에 매크로 실행 (집행내역서 DB 파일)

## 리프레시되는 엑셀 파일 목록
- 양주옥정 집행내역서.xlsx
- 시화호 라군인테라스 집행내역서.xlsx
- 시화호라군인테라스2차 집행내역서.xlsx
- 별내자이 집행내역서.xlsx (2개)
- 창원 상남동 집행내역서.xlsx
- 힐스테이트 장안 집행내역서.xlsx
- 현대프리미어캠퍼스 세마역 집행내역서.xlsx
- 집행내역서(DB).xlsm

## 주요 수정 사항
- 엑셀 리프레시 기능만 남기고 DB 관련 기능은 백업 파일로 분리
- 절대 경로를 상대 경로로 변경하여 이식성 향상
- 파일 경로 오타 수정 (.xlsmm → .xlsm)
